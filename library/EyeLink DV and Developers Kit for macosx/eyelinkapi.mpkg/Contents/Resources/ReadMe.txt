The EyeLink Software distribution for Mac OS X contains the following:
1. Eyelink core libraries and required support libraries
	i. eyelink_core framework
	ii. eyelink_core_graphics framework 
	iii. support libraries for eyelink_core_graphics(sdl, SDL support libraries, and GD for image saving)
	
2. Sample Experiments
	i. Examples using SDL as the graphics platform
	ii. sdl_util (a library to perform gaze contingent cursor and text display.)

3. EDF Access API
	- C, C++, Java EDF Access API and edf2asc as the example
